# Welcome

EtnaUI aims to empower developers and designers to craft swift and accessible digital services using a unified set of UI components, patterns, and templates.

By providing a standardized UI kit, EtnaUI eliminates the need for extensive time investment in building a custom UI kit. This allows developers and designers to concentrate on their projects, ultimately saving time and reducing costs.

We strongly believe in the power of community and open-source projects. Our primary goal is to create a comprehensive UI kit encompassing both web and mobile components, aiming to efficiently assist the open-source community in their project development.

This UI Kit is built by developers and designers for the Design & Development community.



### What’s included?

**Tokens:** A collection of tokens supporting the kit's construction, enabling easy customization without extensive time investment.

**Components:** EtnaUI offers a diverse array of open-source, front-end components that are reusable and customizable. They expedite website development without compromising specific requirements, all rigorously checked for adherence to accessibility standards.

**Blocks:** Assembled multiple components that enhance efficiency and offer a centralized approach.

**Design File for Figma:** A Figma design file is provided, streamlining the design process for designers by eliminating the need to start from scratch.

**React Library:** A comprehensive compilation of front-end components built using the React library.
